# UI设计基础
### UI设计是指对软件的人机交互，操作逻辑，界面美观的整体设计，也叫界面设计
### UI设计师分类
    1.GUI(图形设计师):移动端, 网页, pc端, AR, VR, 车载系统
    
    2.ID(交互设计师):用户操作后的效果设计，界面布局，内容展示，设计思考能力，业务理解能力，复杂的逻辑处理能力，原型图
    
    3.UE/UX(用户体验):用户在使用产品过程中的个人主观感受，关注用户使用前，中，后的整体感受，包括行为，情感，成就等各个方面
### UI设计要做的内容：界面，图标，切图，标注
### 产品开发流程：想法-研究-开发-上线
### 需求分析：商业目标（开发这个产品想要的得到什么），用户需求（用户从这个产品中想要的到什么）
### 原型设计：产品经理-原型图（产品框架和功能），流程图
### 交互Demo: 在原型图的基础上添加一些可互动的交互方式
### 前端开发：产品架构功能编写，页面编写，用户能看到的东西
### 后端开发： 数据接口提供者
### UI设计师：视觉设计-界面设计，切图，标注适配等
### 测试工程师： 产品测试
